# Personal Finance Management System

## Description
This project is a full-stack personal finance management application that helps users track transactions, monitor spending behavior, and generate financial insights.

## Features
- Transaction tracking
- Expense categorization
- Financial dashboard
- Spending insights

## Tech Stack
Frontend: React / React Native  
Backend: Node.js / Spring Boot  
Database: SQL  

## Project Goal
To build a scalable financial tracking system and apply data analysis techniques to generate meaningful financial insights.
